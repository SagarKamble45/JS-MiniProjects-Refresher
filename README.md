# 🟨 JS-MiniProjects-Refresher

> **5 hands-on JavaScript projects built as a revision session before starting React.js**  
> Covers DOM manipulation, event handling, async/await, Fetch API, and more.

---

## 🎯 Why I Built This

Before jumping into React.js, I wanted to make sure my **core JavaScript fundamentals** were solid.  
So instead of just reading theory, I built **5 mini projects from scratch** — each one targeting specific JS concepts.

This is my **pre-React refresher** and the starting point of my frontend journey. 🚀

---

## 📁 Projects Overview

| # | Project | Key Concepts |
|---|---------|-------------|
| 1 | 🎨 Color Generator | DOM manipulation, Random logic, `Math.random()` |
| 2 | 🌡️ Temperature Converter | Functions, Input handling, Conditional logic |

## 🔍 Project Details

### 🎨 1. Color Generator
A simple app that generates a **random hex color** on button click and changes the entire background dynamically.

**Concepts practiced:**
- DOM selection & manipulation
- `Math.random()` for hex color generation
- `onclick` event handling

---

### 🌡️ 2. Temperature Converter
Converts temperature values between **Celsius, Fahrenheit, and Kelvin** instantly.

**Concepts practiced:**
- Input reading & parsing
- Mathematical formula implementation
- Real-time UI update

---

### 📏 3. Unit Converter
A general-purpose converter that handles **common unit types** with a clean interface.

**Concepts practiced:**
- Dropdown/select handling
- Dynamic calculation based on user choice
- Functions & conditionals

---

### ✅ 4. To-Do List
A classic task management app where you can **add, toggle, and remove completed tasks**.

**Concepts practiced:**
- JavaScript Objects & methods
- Array manipulation (`push`, `filter`)
- Dynamic list rendering with `createElement`
- Toggle state management

---

### 📝 5. QuickPost Application *(Most Advanced)*
A mini social-style post app that **fetches real posts from an API**, lets you **create** new ones and **delete** existing posts.

**Concepts practiced:**
- `fetch()` API with `async/await`
- `GET`, `POST`, `DELETE` HTTP methods
- JSON parsing & rendering
- Error handling with `try/catch`
- Dynamic DOM creation & updates

> Uses **JSONPlaceholder** as a mock REST API: `https://jsonplaceholder.typicode.com/posts/`
